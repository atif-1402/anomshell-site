 Introduction

What anomshell is and what to expect from it.

Anomshell is a Quickshell configuration designed specifically for Omarchy setups. It uses Omarchy scripts and tooling and is intended as a personal customization and learning project rather than a general-purpose Quickshell config.


This project depends on Omarchy components. It is mainly intended for Omarchy users and may not work out-of-the-box on other setups.

 What to expect

  • A minimal bar that stays out of your way while remaining informative
  • A fast launcher inspired by the Omarchy Walker menu
  • A minimal dock with smart auto hide and more
  • A Task Manager with lot of Functionality
  • Live settings — no file editing needed for common tweaks
  • Keyboard-first design with dedicated keybinds for everything
  • Fluid OSD with a running cava visualizer in the background
  • Currently in Alpha — expect occasional rough edges
anomshell is built with QML (99.8%) and a small Shell component (0.2%). Actively developed on the main branch.
