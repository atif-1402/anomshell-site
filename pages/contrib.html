 Contributing

How to contribute to anomshell.

anomshell is an open project and contributions are welcome. Since it's primarily a personal config, contributions should align with the project's design philosophy of being clean, minimal, and Omarchy-focused.

1

 Fork the repo

Fork atif-1402/anomshell on GitHub and clone your fork locally.

2

 Make your changes

Work on a feature branch. Keep changes focused and test on an actual Omarchy setup before submitting.

3

 Open a Pull Request

Submit a PR describing what you changed and why. Include screenshots if the change is visual.


This project is in Alpha. Some issues may be closed as "not planned" if they fall outside scope. Please check existing issues before opening a new one.
