 FAQ

Frequently asked questions about anomshell.

 Does anomshell work without Omarchy?

anomshell is designed specifically for Omarchy setups and depends on Omarchy scripts and tooling. It may partially work without Omarchy, but you will need to troubleshoot missing components yourself.
 Why is it in Alpha?

Anomshell started as a personal customization and learning project. While fully functional for daily use, expect occasional bugs or rough edges. Please report issues with logs and screenshots.
 How do I update anomshell?

Pull the latest changes with git pull in the cloned directory, then re-copy the config files as in the install steps. Always backup your existing config first.
 The keybinds aren't working after install.

Make sure ~/.local/bin is in your $PATH, then restart Hyprland. Verify by running quickshell directly from the terminal.
 Can I use a different compositor?

anomshell is built for Hyprland (via Omarchy). Keybinds and certain modules are Hyprland-specific. Using it on other Wayland compositors would require significant rework.
 How do I switch bar position or style?

Open the Settings panel with Super + Ctrl + Alt + Space. Under the "Bar" section you can switch between layouts and positions. Changes apply live instantly.
 How do I report a bug?

Open an issue on the GitHub Issues page. Please include relevant logs and screenshots so the bug can be fixed quickly.
