 Usage

Overview of all anomshell components and how to use them.

  01 / Bar & Dock
Minimal Bar & Smart Dock
A clean, theme-aware bar paired with a smart dock for fast access to apps and system controls. Designed to stay minimal while still giving quick info, notifications, and workflow shortcuts.
  • Omarchy button — opens the main launcher
  • Workspaces — shows workspace list and highlights the active one
  • Media module — shows current track; hover reveals mini player with album-art based colors
  • System stats — CPU and RAM usage with progress preview on hover
  • Tray — clean application tray icons only
  • Notification & Control Center — volume sliders, DND toggle, WiFi and Bluetooth manager
  • Clock — time and date with calendar on hover
  • Smart Dock — quick app launcher with running app indicators
  • Pin apps — right click any app to pin/unpin from the dock
  • Notification dots — small indicators show unread notifications on apps
  • Auto hide — dock hides automatically and appears when needed
  • Customize bar position, style, launcher logo, workspace style, and dock behavior from Settings
  02 / Launcher
App & Action Launcher
A fast, minimal launcher for apps, search, and quick actions. Opens instantly, filters as you type, and stays distraction-free.
  • Global search through all submenus instantly
  • Inspired by the Omarchy Walker menu
  • Quick sections for utilities and custom menus with dedicated keybinds
  • Keyboard-first: type, hit Enter, done
  03 / Theme & Background
Theme & Background Picker
A new interface for picking themes and wallpapers with a clean scrollable layout.
  • Dedicated keybinds for both theme and background
  • One-row scrollable layout
  • Keyboard and mouse navigation supported
  04 / OSD
On-Screen Display
A new-style OSD for volume and media with a fluid cava visualizer running in its background.
  • Dedicated keybinds for each OSD
  • Media OSD shows progress bar and cover art
  • Fluid visualizer animation in background
  05 / Media Card
Media Card
A hoverable media card inspired by end4-dots that adapts colors from album art.
  • Hover the Media module in the bar to access it
  • Play/Pause, Forward and Backward Track controls
  • Adapts to the color of the current Album Cover Art
  06 / Clipboard & Keybinds
Clipboard & Keybinds Manager
A clean manager with organized clipboard history and a keybinds viewer.
  • Dedicated keybinds for both managers
  • Image previews in clipboard with a clear-all button
  • Organized keybinds list viewer
  07 / Settings
Live Settings Panel
A control panel to manage bar style, OSD, launcher icon, workspace style, and behavior — without touching config files. All changes apply live.
  • Bar — choose between layouts and positions
  • Dock — choose behavior
  • Widgets — Enable/Disable the widget and change the format
  • OSD — move the volume/brightness overlay anywhere
  • Launcher — swap the logo/icon to match your theme
  • Workspaces — switch visual styles and numbering
  • General — whole-shell behavior in one UI
  • Changes apply live for instant preview
