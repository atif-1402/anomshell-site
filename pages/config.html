 Configuration

Most configuration is done through the live Settings panel. For deeper tweaks, config files live under ~/.config/quickshell/.


The Settings panel (Super + Ctrl + Alt + Space) handles most common tweaks without touching any files. Use it as your first stop.

 What you can configure via Settings UI

  • Bar layout and screen position (top / bottom)
  • Bar visual style
  • OSD overlay position
  • Launcher logo and icon
  • Workspace numbering and visual style
  • Dock & Widgets behavior
  • General shell behavior

 Manual config files

For deeper customization, the Quickshell config is at:

~/.config/quickshell/

Hyprland keybinds are in:

~/.config/hypr/bindings.conf
